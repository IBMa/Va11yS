ARIA21 - Example 2: Using Aria-Invalid to Indicate An Error Field

Source:

  • ARIA21
  • Forms: Using aria-invalid to identify failed fields

Tested on:

  • Windows 7, Firefox ESR 49.0.1, and JAWS 17
    • Result: Screenreader announces the error messages while tabbing back through the required fields.
  • iOS 11.2.5 Safari browser is based on the iOS version, and VoiceOver on iPad
    • Result - iOS VoiceOver announces:
      • Personal ID, number PIN 4 digits, star, text field, double tap to edit.
      • Email address, star, text field, double tap to edit.
      • Last name, star, text field, double tap to edit.
      • Desired policy start date MM/DD/YYYY text field, double tap to edit.
      • Login button, end, main.
