ARIA20 - Example 2: Using the region role to identify a region of the page

John Henry's Account
  1. Certificate of Deposit:
    Account: 25163522
    Start Date: February 1, 2014
    Maturity Date:February 1, 2016
    Deposit Amount: $ 3,000.00
    Maturity Amount: $ 3,072.43

Source: ARIA20

Tested on:

  • Windows 7, Firefox ESR 49.0.1, and JAWS 17
    • Result: Screenreader announces the error message indicating the number of required fields that are missing data.
  • iOS 11.2.5 Safari browser is based on the iOS version, and VoiceOver on iPad
    • Result - iOS VoiceOver announces:
      • John Henry's Account, button, expanded.
      • Certificate of Deposit.
      • Account: row 1, column 1. 25163522, row 1 column 2.
      • Start Date: row 2, column 1. February 1, 2016 row 2, column 2.
      • Maturity Date: row 3, column 1. February 1, 2016, row 3, column 2.
      • Deposit Amount: row 4, column 1. Three thousand dollars, row 4, column 2.
      • Maturity Amount: row 5, column 1. Three thousand seventy two dollars and fourty three cents row 5, column 2.
