ARIA9 - Example 2: Using aria-labelledby to concatenate a label from several text nodes

Taxpayer Spouse
W2 Gross
Dividends

Source: ARIA9

Tested on:

  • Windows 7, Firefox ESR 49.0.1, and JAWS 17
    • Result: Screenreader announces the column and row number, headings, then edit, type, text for each text field in the table.
  • iOS 11.2 Safari browser is based on the iOS version, VoiceOver on iPad
    • Reult: iOS VoiceOver announces:
      • Taxpayer, row 1 column 2, table start, 3 rows, 3 columns. Spouse, row 1 column 3.
      • W2 Gross, row 2 column 1, Taxpayer W2 Gross, row 2 column 2, text field. Spouse W2 Gross, row 2 column 3, text field.
      • Dividends row 3 column 1, Taxpayer Dividends, row 3 column 2, text field. Spouse, row 3 column 3, text field, table end, end main.
