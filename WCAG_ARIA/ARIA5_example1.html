Tested on:

  • Windows 7, Firefox ESR 49.0.1, and JAWS 17
    • Result: Screenreader announces toggle button pressed bold_label when button is selected and toggle button unpressed bold_label when the button is deselected.
  • iOS 11.0 (15A372), Safari browser is based on the iOS version, VoiceOver on iPad
    • iOS VoiceOver announces:
      • Bold underscore label, toggle button, not pressed. Double tap to toggle settings.
      • Example text 1, when the B is pressed the text changes to bold, multiple line fields, end main. Double tap to edit.
