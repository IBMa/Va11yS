ARIA1 - Example 2: Using aria-describedby to associate instructions with form fields

A bit of instructions for this field linked with aria-describedby.


Source:

  • WCAG ARIA1: Using the aria-describedby property to provide a descriptive label for user interface controls
  • Source code on GitHub

Tested on:

  • Windows 7, Firefox ESR 49.0.1, JAWS 17
    • Result: Screen reader announces First Name, text field, and the descriptive text that follows.
  • Windows 7, Chrome 57, JAWS 17
    • Result: Screen reader announces "First Name, edit" and the descriptive text that follows.
  • Windows 7, IE 11, JAWS 17
    • Result: Screen reader announces "First Name, edit" and the descriptive text that follows.
