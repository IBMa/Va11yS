H45 Example 1: Using longdesc

Source: WCAG H45

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces graphic a complex chart. Press alt + enter for long description.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: There is no way to read longdesc and no way to tell that it is a longdesc. iOS VoiceOver annouces a complex chart image, end main.
