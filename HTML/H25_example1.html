H25 Example 1: Providing a title using the title element

This example is demonstrated by using the title "H25 Example 1" in the head tag.

Source: WCAG H25

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces the title.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces this example is demonstrated by using the title h25 example 1 in the head tag end main.
