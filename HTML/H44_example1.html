H44 Example 1: Using label elements to associate text labels with form controls

Source: WCAG H44

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces first name: edit type in text.
  • Windows 7, Firefox ESR 45.4.0, and NVDA 2016.3
    • Result: Screen reader announces first name: edit.
  • Mac OS X 12.1, Safari, and VoiceOver
    • Result: Screen reader announces first name: edit text.
  • iOS 10.1.1, Safari, and VoiceOver
    • Result: Screen reader announces first name: edit text field.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces first name text field, end main.
