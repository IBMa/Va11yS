H30 Example 3: Providing link text that describes the purpose of a link for anchor elements

Current routes at Boulders Climbing Gym

Source: WCAG H30

Using an empty alt attribute when the anchor element contains text that describes the purpose of the link in addition to the img element. Note that the link text will appear on the page next to the image.

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces link text.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces current routes at boulders climbing gym link, end main.
