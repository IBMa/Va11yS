H59 Example 1: Using the link element and navigation tools

Source: WCAG H59

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: All link elements pertaining to navigation occur in the head section of the document. Each link element in the head section of the document which pertains to navigation has a rel attribute identifying the link type and a valid href attribute to locate the appropriate resource.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces 3 links.
      • Home link, navigation landmark.
      • Greater than.
      • HTML Techiques link end, navigation.
