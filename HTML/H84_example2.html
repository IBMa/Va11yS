H84 Example 2: Using a button with a select element to perform an action

Source: WCAG H84

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader annouces combo box and value. Context does not change until the Do It button is selected.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver annouces
      • Options, help popup button, double tap to activate the picker.
      • Do it button, end, main
