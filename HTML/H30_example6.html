H30 Example 6: Providing link text that describes the purpose of a link for anchor elements

MyCorp 2009 Annual Report (pdf)
MyCorp 2009 Annual Budget (Excel)

Source: WCAG H30

The “MyCorp” company’s annual report is made available on the corporate website as a PDF file, and the annual corporate budget is made available as an Excel file on the web site.

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces link text.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces
      • MyCorp 2009 annual report pfd link.
      • MyCopr 2009 annula bugget exel link, end main.
