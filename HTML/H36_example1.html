H36 Example 1: Using alt attributes on images used as submit buttons

Source: WCAG H36

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces Submit button when image has focus.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces submit button, end main.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces image as submit button. Note: will not reads the alt attribute submit button when focus on the image. Only announce via the Global content menu.
