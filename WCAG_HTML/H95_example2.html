H95 Example 2: Using the track element to provide captions

Source: WCAG H96

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Captions display in the default language, which is English.
    • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
      • Result: Captions display in the English text on the video.
    • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
      • Result: Captions display in the video.
