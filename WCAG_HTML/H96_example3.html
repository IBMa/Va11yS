H96 Example 3: Using the track element to provide audio descriptions

Source: WCAG H96

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces audio descriptions
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: Captions display in English as default language.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces audio descriptions.
