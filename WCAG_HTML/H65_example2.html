H65 Example 2: Using the title attribute to identify form controls when the label element cannot be used

Phone number

Source: WCAG H65

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces area code edit, type text. First 3 didgits of phone number, type text. Last 4 digits of phone number, type text.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces
      • Area code text field.
      • First 3 digits of phone number text field.
      • Last four digits of phone number text field, end phone number, end main, form end.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces
      • Edit box, area code. Tap to enter text.
      • Edit box, first 3 digits phone number. Tap to enter text.
      • Edit box, last 4 digts phone number. Tap to enter text.
