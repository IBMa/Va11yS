H51 Example 1: A schedule marked up as a simple data table with column and row headers

Monday Tuesday Wednesday Thursday Friday
8:00-9:00 Meet with Sam

Source:

    • Tested on:

    • Windows 7, Firefox ESR 45.4.0, and JAWS 17
      • Result: Screen reader announces the content of the data table.
    • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
      • Result: iOS VoiceOver did not read content as expected when using swipe left/right gesture. Column headings read out of order. VoiceOver announces:

        • Wednesday Tuesday colum3.
        • Thursday Wednesday column4.
        • Friday Thursday column5.
        • Friday column6.
        • Monday 8 9 row2 column1.
        • Tuesday meet with Sam column2.
        • Monday 9 10 row3 colum1.
        • Thursay doctor Williams column4.
        • Friday Sam again column5.
        • Leave for San Antonio column6 table, end, end, end main.
      • Result: iOS VoiceOver reads as expected when using 2 finger swipe down Read All gesture.
        • Monday row1 colum2, table start 3 rows, 6 colums.
