H97 Example 1: Grouping related links using the nav element

Web Accessibility | Document Accessibility | Mobile Accessibility

Source: WCAG H97

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces navigation region and the link text for each link.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces
      • Web accessiblity link, secondary, navigation landmark.
      • Document accessiblity link.
      • Mobile accessibility link end, secondary navigation end main.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces
      • Web accessibility. Tap to activate.
      • Document accessibility. Tap to activate.
      • Mobile accessibility. Tap to activate.
