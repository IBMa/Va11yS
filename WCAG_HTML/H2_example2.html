H2 Example 2: Combining adjacent image and text links for the same resource

Go to the home page

Source: WCAG H2 A link contains an icon and text, and the site help refers to the icon. The img has a text alternative which is the name used for the icon in the site help, which describes clicking the home page icon.

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Announces home page icon, Go to the home page, link graphic.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces homepage icon go to home page link, end main.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces home page icon go to home page. Tap to activate.
