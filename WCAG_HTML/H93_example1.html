H93 Example 1: Ensuring that id attributes are unique on a Web page

Fruits:
  • Apple
  • Orange
  • Pear
Vegetables:
  • Peas
  • Carrots
  • Spinach

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17.
    • Result: HTML validation does not find duplicate ids.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: HTML code shows ids are unique id="fruits" for Fruits and id="vegetable" for Vegetable in the list on the web page.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces
      • Fruits, tap to activate.
      • List showing three items.
      • Orange, tap to activate.
      • Pear, tap to activate.
      • Vegetables, tap to activate.
      • List showing three items, tap to activate.
      • Peas, tap to activate.
      • Carrots, tap to activate.
      • Spinach, tap to activate.
