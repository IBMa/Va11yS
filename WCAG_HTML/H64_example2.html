Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: After navigating to the document using iframe link, screen reader announces advertisement frame banner.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: After actvating the document using the iframe link. iOS VoiceOver announces IBM banner image, banner landmark.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: After activating the a document using the iframe link, TalkBack announces content in frame "IBM banner".
