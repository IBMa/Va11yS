H84 Example 2: Using a button with a select element to perform an action

Source: WCAG H84

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces combo box and value. Context does not change until the Do It button is selected.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces
      • Options, help pop-up button "help button selected by default on the combo box list, double tap to activate the picker.
      • Do it button, end main. After Do it button is selected.
      • An alternate dialog pop-up with the message iOS VoiceOver announces: "This form is not secure. Are you sure you want to submint it?" Cancel button, Submit button, "Select submit button, there is new page pop-up"
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces
      • Collapsed, help, dropdown list. Tap to change.
      • Do it button. Tap to activate.
