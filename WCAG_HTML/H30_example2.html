H30 Example 2: Providing link text that describes the purpose of a link for anchor elements

Source: WCAG H30

Using the alt attribute for the img element to describe the purpose of a graphical link.

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces alt text.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces current routes at boulders climbing gym link image, end main.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces current routes at boulders climbing gym. Tap to activate.
