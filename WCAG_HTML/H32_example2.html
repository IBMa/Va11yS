H32 Example 2: Providing submit buttons





Source: WCAG H32

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces navigate the site, Combo box, home. To change the selection use the arrow keys. Context changes when the Go to Page button is selected.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces
      • Navigate the site home pop-up button. double-tap to activate the picker, "Home" is selected on the combo box. To change the selection use the figure flick up or down on the screen. Redirects the user to the requested page (Example Domain page)
      • Go to page button, end main.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces
      • Navigate the site.
      • Collapse home drop down list. Tap to change.
      • Go the page button. Tap to activate.
