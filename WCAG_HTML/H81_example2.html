H81 Example 2: Identifying the purpose of a link in a nested list using link text combined with the parent list item under which the list is nested

  • Hotel Three Rivers

Source: WCAG H81

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces link text for each item on the list.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces
      • Royal Palm Hotel link, list start.
      • Map link.
      • Photos link.
      • Directions link.
      • Guest reviews link.
      • Book now link, list end.
      • Hotel Three Rivers link.
      • Map link.
      • Photos link.
      • Directions link.
      • Guest reviews link.
      • Book now link, list end, end, end main.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces
      • Royal Palm Hotel, in list, level 1, 2 items. Tap to activate.
      • Map, in list level 2, 5 itemas. Tap to activate.
      • Photos. Tap to activate.
