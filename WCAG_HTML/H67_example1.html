H67 Example 1: Using null alt text and no title attribute on img elements for images that AT should ignore

Source: WCAG H67

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader ignores the graphic.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver skips the graphic on the user interface.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces the graphic on the user interface. Note: No way to access the image by touch screen or TalkBack accessility features.
