H65 Example 1: Using the title attribute to identify form controls when the label element cannot be used

Source: WCAG H65

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces search for: edit, type text. Search in combo box to make a select use the arrow keys.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces
      • Search for text field.
      • Search in books popup button, end main.
      • Book selected in the pop-up combo box.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces
      • Edit box, search for. Tap to enter text.
      • Collapsed dropdown list. Tap to change.
      • Showing items, 1 to 3 of 3.
