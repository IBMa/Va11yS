  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces

      • White bullet, list start, HTML link.
      • White bullet, PDF link.
      • White bullet, RTF link, list end.
      • Annual Report 2 0 0 6 2 0 0 7.
      • White bullet, list start, HTML link.
      • White bullet, PDF link.
      • White bullet, RTF link, list end, end, end, end main.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.

      • PDF.

      • Dot, in list level 1, 2 items, annual report 2 0 0 6 2 0 0 7.
