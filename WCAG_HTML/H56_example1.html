H56 Example 1: Using the dir attribute on an inline element to resolve problems with nested directional runs

The title says "פעילות הבינאום, W3C" in Hebrew.

Source: WCAG H56

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces the English and Hebrew text.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces the title says פעילות הבינאום in Hebrew.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces the title says WC3C in Hebrew. Note: TalkBack did not read the Hebrew letters.
