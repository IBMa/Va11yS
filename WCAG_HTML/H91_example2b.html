H91 Example 2b: Using HTML form controls and links

Source: WCAG H91

Tested on:

  • Windows 7, Firefox ESR 45.4.0, and JAWS 17
    • Result: Screen reader announces save button, submit button, and reset button.
  • iOS 10.2.1, Safari browser is based on the iOS version, VoiceOver on iPad
    • Result: iOS VoiceOver announces save button, submit button, reset button, end main.
  • Android 7.1.2; Pixel, Chrome 55.0.2883.91, and TalkBack Version: 5.0.4.3.
    • Result: TalkBack announces
      • Save button. Tap to activate.
      • Submit button. Tap to activate.
      • Reset button. Tap to activate.
