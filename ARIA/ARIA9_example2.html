    • Result: Screenreader announces the column and row number, headings, then edit, type, text for each text field in the table.
  • iOS 11.2 Safari browser is based on the iOS version , VoiceOver on iPad
    • iOS VoiceOver announce
      • Taxpayer, row1 column2, table start, 3rows, 3columns. Spouse, row1 column3.
      • W2 Gross, row2 column1, Taxpayer W2 Gross, row2 column2, text field. Spouse W2 Gross, row2 column3, text field.
      • Dividends row3 column1, Taxpayer Dividends, row3 column2, text field. Spouse, row3 column3, text field, table end, end main.
