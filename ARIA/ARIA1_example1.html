Tested on:

  • Windows 7, Firefox ESR 49.0.1, JAWS 17
    • Result: Screen reader announces close button and the descriptive text that follows.
      Note, close window button does not close the window on Firefox ESR 49.0.1.
  • Windows 7, Chrome 57, JAWS 17
    • Result: When focus lands on button, JAWS reads "Close, button." then the descriptive text.
      Note: close button does not close the window.
  • Windows 7, IE 11, JAWS 17
    • Result: When focus lands on button, JAWS reads "Close, button." then the descriptive text.
